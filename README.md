# Muzei Roma

Sorgente di wallpaper per [Muzei](https://github.com/muzei/muzei) dedicata all'arte di Roma.

Mostra dipinti, sculture e vedute provenienti dai grandi musei romani — Galleria Borghese, Musei Capitolini, Musei Vaticani, Palazzo Barberini, Galleria Doria Pamphilj — con immagini in alta risoluzione da Wikimedia Commons.

## Funzionamento

1. Installa [Muzei](https://play.google.com/store/apps/details?id=net.nurik.roman.muzei) sul dispositivo
2. Installa Muzei Roma
3. In Muzei → seleziona **Muzei Roma** come sorgente
4. Le immagini vengono scaricate in background e ruotate automaticamente come wallpaper

## Caratteristiche

- Archivio di oltre 30 opere d'arte romane
- Download locale delle immagini (4K/1920px da Wikimedia Commons)
- Aggiornamento automatico ogni 3 ore via WorkManager
- Metadati completi: autore, data, tecnica, collocazione
- Link diretto alla pagina Wikipedia dell'opera
- Nessun account, nessuna API key necessaria

## Requisiti

- Android 8.0+ (API 26)
- App [Muzei](https://play.google.com/store/apps/details?id=net.nurik.roman.muzei) installata

## Sorgenti immagini

Tutte le immagini provengono da [Wikimedia Commons](https://commons.wikimedia.org) e sono utilizzate secondo le rispettive licenze Creative Commons.
